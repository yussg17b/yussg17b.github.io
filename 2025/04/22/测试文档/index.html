在 Hexo 中更改主页图片，你需要修改主题的配置文件。具体步骤如下：

1. 找到主题配置文件:

  • Hexo 主题通常位于 themes/<你的主题名称> 目录下。 例如，如果你使用的是默认的主题 “hexo”，那么文件路径就是 themes/hexo。
  • 在主题目录中，最关键的配置文件是 config.yml。 这个文件包含了主题的所有设置选项。

2. 编辑 config.yml 文件:

  • 使用你喜欢的文本编辑器打开 config.yml 文件。
  • 找到 coverImage 选项。 它通常位于 theme: 部分，或者在主题的全局配置部分。
  • 修改 coverImage 的值，将其设置为新的图片 URL。 URL 可以是本地路径或远程 URL。

  • coverImage: 这个选项指定了主页的背景图片。
  • /images/my-new-hero-image.jpg: 这是一个示例路径。 你需要将 /images/my-new-hero-image.jpg 替换为你想要使用的图片的实际 URL 或本地文件路径。

3. 保存并刷新:

  • 保存 config.yml 文件。
  • 在 Hexo 中运行命令 hexo s (或 npm run dev) 来重新编译和启动服务器，以便应用新的配置。 这会更新你的主页，显示你设置的新图片。

其他注意事项：

  • 本地文件路径: 如果你使用本地文件路径，确保图片文件位于主题的 static/images 目录下（或者你自定义的静态资源目录）。
  • 远程 URL: 如果使用远程 URL，请确保该 URL 是有效的并且可以访问。
  • 图片尺寸: 建议选择合适的图片尺寸，以避免主页加载速度过慢。 你可以使用图像编辑工具调整图片的尺寸。
  • 主题兼容性: 某些主题可能对 coverImage 的设置有特殊要求。 请查阅你所使用的主题的文档，了解更多信息。
