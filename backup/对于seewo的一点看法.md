# 希沃（seewo）
设备解除冰点还原一般不建议直接通过更改注册表的方式，因为操作不当可能会导致系统不稳定或出现其他问题，但如果一定要通过注册表来解除冰点还原，以冰点还原 6.0 为例，大致步骤如下：
准备工作：如果电脑是单系统，需要准备深山红叶 PE 等可从光盘或 U 盘启动的操作系统，以及 Registry Workshop 等注册表编辑工具；如果是双系统，可以直接使用系统自带的注册表编辑器（regedit）。
# 进入可编辑注册表的环境：
若使用单系统，从光盘或 U 盘启动进入深山红叶 PE 系统；若是双系统，可直接进入非冰点保护的系统。
# 打开注册表编辑器并加载配置单元：
使用 Registry Workshop 时，从文件菜单中选择 “加载配置单元”，定位到安装了冰点还原的系统盘的 C:\WINDOWS\system32\config 路径，找到 system 文件并打开，项名称可随意填写，如 “df”，加载位置默认即可。如果使用系统自带的注册表编辑器，直接打开注册表编辑器（regedit）。
# 查找并删除相关项：
在注册表中查找所有包含 “deep” 字样的内容。一般来说，在 HKEY_LOCAL_MACHINE\df 项中，红线以上的部分是冰点的服务项，将这些项删除。
# 编辑 UpperFilters 值：
红线以下的部分需要逐项编辑其 UpperFilters 值，清除数值数据中的 “DeepFrz” 字样。也可一次性导出用记事本编辑好后再导入。注意删除 “DeepFrz” 后不能留有空行。
# 卸载配置单元并重启：
对每一项 UpperFilters 值都编辑好之后，将鼠标定位到 “df” 项上，然后从文件菜单中选择 “卸载配置单元”。重新启动进入原来安装了冰点还原的硬盘操作系统，此时冰点还原应该已被解除。
